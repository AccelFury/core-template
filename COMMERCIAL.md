# Commercial Licensing

This template is released for open collaboration under these files:

- `LICENSES/CERN-OHL-S-2.0.txt`
- `LICENSES/AGPL-3.0-or-later.txt`
- `LICENSES/CC-BY-SA-4.0.txt`

Open-source terms apply by default. For exceptions:

- proprietary commercial license for closed-source FPGA bitstreams,
- ASIC integration support,
- paid prover service,
- commercial SaaS usage.

If open-source copyleft terms are not acceptable, separate written commercial
terms are required before using this work in a proprietary product.

Attribution is required through `NOTICE` and contribution process requires
`CLA.md`.
